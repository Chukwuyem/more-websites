Nigeria project

Menu

The Nigeria project is an amalgam of multiple ideas. One of these was an idea for a website to show the stories of experiences of the common person in a particular historical event. The idea was based on the assumption that these stories and experiences, that would be gathered manually, would be readily available. However, the difficult of finding these stories was an obstacle.

Later on, still working on the same idea, the creator came across ACLED data, a project that collected and aggregated data on conflict and war events, most of which were occurrences that barely made any news or got any form of recognition. This was similar to the previous idea of experiences and stories that would usually not get told. This is especially because a lot of these occurrences were skirmishes, a good amount of which had no fatalities, but were still politically relevant. The creator couldn't help but think of a way to mutate the first idea to work with ACLED's data. This result: NIGERIA PROJECT.

Furthermore down is a more detailed "manual" of sorts on how NIGERIA PROJECT works and how it is expected to be used. But I suspect that if you're reading this, it's too late. You've already figured out how to use it.

I do, however, still want to point of one thing - what makes NIGERIA PROJECT much more than a data visualization project. Think of the first idea mentioned: personal stories of events. With NIGERIA PROJECT, we have these events, ranging from the very small skirmishes to clashes with fatalities in the thousands. Moreover, in NIGERIA PROJECT, there is input. A user going through the events and sees one which they experienced can add a short comment or a detailed narrative of their experience of that event, which is published and associated with that event. Another user can look at that same event and see the first user's story.
